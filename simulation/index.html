Industrial Simulation

One of my other key interests is simulation (3D or otherwise) for industrial control systems. It's a fasincating way to learn about the ins-and-outs of a control system before actually seeing it in the field.

During my career I have made many detailed machine simulations using the Ignition platform:

Simulations are invaluable for flushing out control system design errors, stepping through complicated sequences, and polishing operator interfaces.

Here are some advantages of using Ignition as a simulation platform:

  • Design and develop simulation interface using standard Ignition Perspective/Vision features
  • Communicate to variety of PLCs via built-in drivers
  • Use of direct OPC reads/writes + SFCs for round-trip communications as fast as 50ms
  • Use visual templates and UDTs for ease of creation

Scada Sim 3D

An ongoing project to connect the Ignition (SCADA) and Godot (3D) technologies for the purposes of simulation and 3D visualization.

Some high level details of the project:

  • Leverage Ignition's OPC-UA architecture to talk to many different hardware platforms
  • Optimize communication via direct OPC reads/writes with Ignition's SFC library
  • A customized Ignition add-on which connects the Ignition server to a Godot 3D game instance, allowing for Ignition tag configuration
  • Use of the Godot 3D Game Engine for developing physical simulations, with the Jolt Physics Engine
