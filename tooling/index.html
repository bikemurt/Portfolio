Game Engine Tooling

Creating tooling for game engines and 3D applications is something we have a passion for. Being able to build a tool that helps you remove the pain and suffering from game development is (almost) always worth it!

  • Blender-Godot 4 Pipeline: a tool that enables you to encode all kinds of collisions, properties, game logic, and so on, into 3D objects inside Blender.
  • Godot 4 Vertex Painting Tool: A tool to... paint vertices!
  • Vertex Color Pipeline ("V Pipeline"). Check out the tech demo videos here, as well as the git repo. Heavily inspired by The Ascent GDC talk.
