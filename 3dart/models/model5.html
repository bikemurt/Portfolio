Show Workflow

  • Poly modeled several distinct components using a boolean/bevel workflow.
  • Auto smooth + harden normals to create clean shading with minimal geometry.
  • Mark seams, UV unwrap and rescale islands as appropriate.
  • Merge separate objects pieces with custom-split normals enabled on all objects to maintain shading.
  • Place UV islands onto single object UV map.
  • Develop layered shader in Blender to allow for multiple layers of paint (alpha blending) and stencils
  • Bake to single texture set using SimpleBake
