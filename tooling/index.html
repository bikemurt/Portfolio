Game Engine Tooling

Creating tooling for game engines and 3D applications is something I have a passion for. I think it's for the same reason people love city builders: you learn how to do a thing, and then optimize the heck out of it.

  • Blender-Godot 4 Pipeline: a tool that enables you to encode all kinds of collisions, properties, game logic, and so on, into 3D objects inside Blender.
  • Godot 4 Vertex Painting Tool: As the name implies
  • Vertex Color Pipeline (dubbed "V Pipeline"). Check out the tech demo videos here, as well as the git repo. Heavily inspired by The Ascent GDC talk.
