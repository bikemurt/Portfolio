Industrial Simulation

The area I have worked in the most as an engineer is in simulation (3D or otherwise) for industrial control systems. Industrial control simulations (sometimes called "digital twins") require real-time, physically accurate simulations. There are a number of important benefits in designing such systems:

  • Iterate and test development ideas in a risk free environment
  • Communicate and demonstrate the success of controls software to stakeholders
  • Educate junior engineers on processes in a safe environment

Scada Sim 3D

This is an ongoing project. Please reach out if you would like to collaborate. Project goals:

  • Use the Godot Game Engine to produce physically accurate, 3D simulations
  • Extend the Open Industry Project
  • Design and develop simulation interfaces using Ignition HMI software
  • Communicate to variety of PLCs via built-in drivers
  • Use of direct OPC reads/writes + SFCs for round-trip communications (~50ms)
  • Develop a customized Ignition add-on which connects the Ignition server to a Godot 3D game instance, allowing for Ignition tag configuration
